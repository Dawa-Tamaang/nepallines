To create a linear gradient in React Native with 70% white and the other color specified in your colors array, you can use the LinearGradient component from the expo-linear-gradient package. Here's how you can achieve it:

First, make sure you have expo-linear-gradient installed in your project:

bash
npm install expo-linear-gradient

Then, you can use it in your React Native component:

javascript
import React from 'react';
import { View } from 'react-native';
import { LinearGradient } from 'expo-linear-gradient';

const YourComponent = () => {
  const colors = ["#005AA7", "white"];
  return (
    <View style={{ height: "100%" }}>
      <LinearGradient
        colors={["rgba(255, 255, 255, 0.7)", colors[0]]}
        style={{ flex: 1 }}
        start={{ x: 0, y: 0 }}
        end={{ x: 1, y: 0 }}
      />
    </View>
  );
};

export default YourComponent;

In the colors prop of the LinearGradient component, the first color is set to rgba(255, 255, 255, 0.7), which is white with 70% opacity. The second color is taken from your colors array. Adjust the opacity percentage as needed.
